ersionamento funciona atraves do git. Ele é o versionador, e ele funciona assim: primeiro criamos um diretorio com 'mkdir', depois disso digitamos um comando 'git init' que é o que inicia o versionamento. Após isso criamos uma branch ou ramo que é como se fosse o nosso rascunho, que é onde podemos fazer alterações dos nossos projetos sem interferir no original ou Main. Essa branch é feita atraves do comando 'git checkout -b e nome da branch' após isso faz as alterações que estiver buscando. E então você irá colocar as suas alteraçoes em modo stage com o comando 'git add . ou git add e nome do arquivo', depois voce usa o comando 'git commit -m '' ' para deixar tudo preparado para o merge, que é o irá mesclar o main com o seu brachn com as alterações. entao voce volta para o branch main com o comando 'git checkout main' e entao voce faz o comando 'git merge e nome da branch com as alterações'.
